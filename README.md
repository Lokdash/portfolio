# portfolio

Для сборки и запуска используется приложение https://github.com/boxfuse/boxfuse-sample-java-war-hello

1. Скрипт на bash, проверяющий превышение квоты использования диска пользователями ("Сценарии командной оболочки. Linux, OS X и Unix. 2-е издание", пример 35). 
2. Скрипт-обертка на bash, протоколирующий удаление файлов пользователями. При вызове с флагом -s логирование не осуществляется ("Сценарии командной оболочки. Linux, OS X и Unix. 2-е издание", пример 17).
3. Скрипт на Python для проверки доступности сервера. Ежеминутно посылает запрос и отправляет оповещение в телеграм при получении ответа, отличного от 200 ОК.
4. Файл Docker для сборки и запуска простого веб-приложения 
5. Ansible-playbook для сборки и запуска простого веб-приложения  на отдельных группах хостов
6. Файл docker-compose для сборки и запуска простого веб-приложения (порт 8083)
